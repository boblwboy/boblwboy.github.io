Questions and Answers

发表于 2014-03-16 | 分类于 Docs
Frequently asked questions and my answers.
What does Freemind stands for?

Freemind is named after Pluskid’s blog. This theme is greatly inspired by his blog layouts and stylesheets.

There’re already so many themes for Hexo. Why create this one?

In fact I created this theme before I turned to use Hexo. I switched to Hexo about half an year ago, including my theme, since I have already got used to it and don’t wanna change it for now.

Why use raw CSS stylesheets instead of more fashionable stylus?

Yes, stylus is cool. But I don’t want to wrote my stylesheets again in stylus. So let’s just keep them.

I love your theme. How to contribute?

Great that you love it. To devote your contribution, you can:

  • Star its Github project;
  • Fork this project, make your change, and then send me your pull request;
  • Since it is public under MIT license, you can make your own theme based on mine. But it will be nicer if you claimed that your work is based on mine in your theme project page.

Your blog looks a little different to this theme. Why such difference?

Yes. I modified the theme a little bit as I open sourced this theme. I actually did some simplification e.g. I removed the whole Wiki page because I don’t think that everybody need this.

How to generate ToC(Table of Contents) in a certain page?

Add toc: true in the front-matter.

Where can I find your markdown source files of these docs?

In the source branch.
